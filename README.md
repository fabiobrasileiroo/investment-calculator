\[

### **O CDI é fixo ou depende do banco?**
O CDI é um índice fixo, calculado diariamente pela B3, e não depende do banco. O que o banco faz é oferecer **um percentual do CDI** (como 110%, 120%, etc.) para atrair investidores.
